# 消消乐 H5 游戏 - 产品需求文档 (PRD)

## 1. 项目概述

### 1.1 项目背景
开发一款基于 H5 的三消类休闲游戏，支持移动端访问，具备完整的关卡系统、特殊元素机制和生成式音效。

### 1.2 目标用户
- 休闲游戏玩家
- 移动端用户（iOS/Android 浏览器）
- 年龄层：全年龄段

### 1.3 核心体验
- 简单易上手的操作
- 爽快的消除反馈
- 策略性的特殊元素组合
- 渐进式难度曲线

---

## 2. 核心玩法

### 2.1 基础规则
- **棋盘尺寸**：6×6 网格
- **元素类型**：6 种不同颜色/图案的基础元素
- **操作方式**：点击或滑动交换相邻两个元素
- **消除条件**：3 个或以上同色元素横向或纵向连成一线

### 2.2 重力系统
- 消除后，上方元素自动下落填补空位
- 顶部生成新元素补充棋盘
- 下落过程触发新的消除判定（连锁反应）

### 2.3 交互流程
1. 玩家选择第一个元素（高亮显示）
2. 玩家选择相邻的第二个元素（或滑动方向）
3. 系统判定是否可交换（交换后能否形成消除）
4. 若可消除：执行交换 → 播放动画 → 消除 → 下落 → 判定新消除
5. 若不可消除：交换回退，提示无效操作

## 3. 特殊元素系统

### 3.1 特殊元素生成规则

| 触发条件 | 生成元素 | 视觉标识 | 效果 |
|---------|---------|---------|------|
| 4 连消（横向） | 横向炸弹 | 元素 + 横向条纹 | 消除整行 |
| 4 连消（纵向） | 纵向炸弹 | 元素 + 纵向条纹 | 消除整列 |
| T 型消除（5 个） | 范围炸弹 | 元素 + 爆炸图标 | 3×3 范围爆炸 |
| L 型消除（5 个） | 范围炸弹 | 元素 + 爆炸图标 | 3×3 范围爆炸 |
| 5 连消（直线） | 彩虹球 | 彩虹色球体 | 消除全屏同色元素 |

### 3.2 特殊元素激活方式
- **直线炸弹**：与相邻元素交换，或作为消除的一部分被触发
- **范围炸弹**：同上
- **彩虹球**：与任意元素交换，消除全屏该颜色所有元素

### 3.3 特殊元素组合（进阶）
- 两个直线炸弹交换：十字消除（行+列）
- 范围炸弹 + 直线炸弹：3 行 + 3 列消除
- 两个范围炸弹交换：5×5 大范围爆炸
- 彩虹球 + 特殊元素：全屏该颜色元素变为对应特殊元素并触发

## 4. 关卡系统

### 4.1 关卡类型

#### 4.1.1 限步模式 (Move Limit)
- 玩家在限定步数内完成目标
- 步数耗尽时未达标则失败
- 消除目标元素或清除障碍物消耗步数

#### 4.1.2 清除模式 (Clear Mode)
- 棋盘上有特定障碍物（冰块、果冻、石块等）
- 目标：清除所有指定障碍物
- 障碍物需要相邻消除或特殊元素才能清除

### 4.2 MVP 关卡设计（5 关）

| 关卡 | 类型 | 步数限制 | 目标 | 特殊机制 |
|-----|------|---------|------|---------|
| 1 | 限步 | 20 | 消除 30 个红色元素 | 教学关，仅基础消除 |
| 2 | 限步 | 25 | 消除 20 个蓝色 + 20 个绿色 | 引入多目标 |
| 3 | 清除 | 30 | 清除 8 个冰块 | 引入障碍物 |
| 4 | 混合 | 35 | 清除 5 个果冻 + 消除 25 个黄色 | 组合目标 |
| 5 | 限步 | 25 | 达到 5000 分 | 分数挑战 |

### 4.3 障碍物设计

| 障碍物 | 清除方式 | 特性 |
|-------|---------|------|
| 冰块 | 相邻消除 1 次 | 单层，透明外观 |
| 果冻 | 相邻消除 2 次 | 双层，颜色变化提示剩余层数 |

## 5. 计分系统

### 5.1 基础分数
- 3 连消：30 分/元素
- 4 连消：40 分/元素
- 5+ 连消：50 分/元素

### 5.2 加成系数
- 连锁反应：第 N 次连锁 ×N 倍
- 特殊元素消除：×2 倍
- 组合特效：×3 倍

### 5.3 星级评定
- 1 星：完成关卡目标
- 2 星：达到目标分数的 150%
- 3 星：达到目标分数的 200%

## 6. 音效系统

### 6.1 技术方案
- **引擎**：Web Audio API
- **方式**：程序化生成（不依赖外部音频文件）
- **库**：使用 Web Audio API 原生或 Tone.js 简化开发

### 6.2 音效清单

| 事件 | 音效描述 | 生成参数 |
|-----|---------|---------|
| 选中元素 | 短促高频音 | 正弦波，800Hz，0.1s |
| 交换元素 | 滑动音效 | 频率从 400Hz 滑到 600Hz，0.15s |
| 基础消除 | 清脆"叮"声 | 三角波，600Hz，0.2s |
| 连锁消除 | 音调递增 | 每次连锁 +100Hz |
| 特殊元素生成 | 华丽音效 | 和弦，多振荡器 |
| 炸弹爆炸 | 低频爆破感 | 锯齿波 + 噪声，快速衰减 |
| 彩虹球激活 | 魔法音效 | 高频琶音 |
| 胜利 | 胜利和弦 | C大调和弦上行 |
| 失败 | 低沉提示 | 低频，渐弱 |

### 6.3 音量控制
- 主音量开关
- 音效音量滑块（默认 70%）
- 无背景音乐
